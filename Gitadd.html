GITADD

$ git add

Команда добавляет файл(ы) в область staged

  1. $ git add file.txt => добавить только file.txt
  2. $ git add folder => добавить папку folder и все содержимое папки
  3. $ git add –A => с флагом –A добавит все файлы в область staged
  4. $ git add . => более сокращенный аналог команды выше, добавит все файлы по текущему пути (.)
